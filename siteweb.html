Application Web "La Fourchette Rurale"

Voici mon 2ème projet que j'ai réalisé dans le cadre de ma formation d'Analyste Informaticien - Concepteur Développeur d'Applications à l'école EQL

Il s'agit d'une application Web JEE utilisant JavaScript, HTML/CSS, une base de données MySQL ainsi qu'un déploiement vers le serveur JBoss EAP

La base de données regroupe les adhérents et les livreurs de l'association "La Fourchette Rurale", lorsqu'un utilisateur crée un compte sur l'application, toutes les informations le concernant sont enregistrées dans la base de données

L'application Web permet aux adhérents, une fois qu'ils auront crée leur compte sur le site, de commander des repas qu'ils auront choisis selon les menus disponibles, ils pourront également modifier leurs informations et payer leurs commandes directement sur le site

Les livreurs pourront également se connecter sur l'application afin de créer/modifier leurs tournées de livraison

Ce projet a été réalisé par une équipe agile de 4 personnes à partir d'un cahier des charges dans un sprint de 2 semaines

Technologies:
