Annuaire des stagiaires EQL

Voici un projet que j'ai réalisé dans le cadre de ma formation d'Analyste Informaticien - Concepteur Développeur d'Applications à l'école EQL

Il s'agit d'un annuaire utilisant JavaFX et regroupant des stagiaires issus de différentes promotions de l'école EQL

La liste de stagiaire peut être importé dans l'application depuis un fichier texte et ses données sont triés par ordre alphabétique en utilisant la structure d'arbre binaire

L'application permet de visualiser les promotions, rechercher les stagiaires, exporter la liste de stagiaires en format PDF, mais également de gérer les administrateurs et de créer une documentation pour aider les nouveaux utilisateurs à prendre en main le site

Ce projet a été réalisé par une équipe agile de 4 personnes dans un sprint de 2 semaines

Technologies:
  • - Java SE
  • - JavaFX
  • - Intelij
  • - GitHub
